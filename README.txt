NEURAL GAS ALGORITHM - a clustering algorithm for OpenCV
////////////////////////////////////////////////////////////////////////////

Discussions can be started in three ways:
- Go to http://answers.opencv.org/question/17461/help-with-clustering-algorithm-called-neuralgas/ and place it there.
- Create an issue on the active repository, and discuss problems there.
- Create a pull request on top of the repository.
